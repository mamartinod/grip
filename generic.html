Present for python 3 compatibility.

This flush method is needed for python 3 compatibility. This handles the flush command by doing nothing. You might want to specify some extra behavior here.

generic.tempering(func, tempering_factor)

Scale a function by a constant. It performs tempering in MCMC, i.e. to smoothen/sharpen the log-likelihood function. Indeed, if the log-likelihood decrease by 1 unit, it means the event is 2.7x less likely to happen. Some log-likelihood functions needs to be tempered before being explored by MCMC algorithm.

Note: the posterior is widen/shrinked by the square root of the tempering factor \(1 / \sqrt{tempering~factor}\).

Parameters

funccallable

Function to rescale.

tempering_factorfloat

Scale factor.

Returns

callable

Rescaled function.

>>> tempering(log_chi2, -2 / ddof) # Returns a reduced chi2 cost function
